# 鸡汤文章写作风格指南

> **风格来源**：洞见公众号（DJ00123987）  
> **分析样本**：3篇文章  
> **风格定位**：深度、温暖、有力、共鸣  
> **适用场景**：人生哲理、家庭教育、成长励志类文章

---

## 🎯 核心写作原则

### 风格关键词
**深度、温暖、有力、共鸣**

### 价值观
- 正向积极的人生态度
- 普世价值（家庭、教育、成长）
- 深度思考，不浮于表面

---

## 📝 标题设计公式

```
标题 = 【时机/场景】+ 【痛点/矛盾】+ 【解决方案/观点】
```

### 标题类型

1. **对比式**
   - 穷人的富养 vs 富人的富养
   - 低谷期 vs 改命期

2. **数字式**
   - 3个能力
   - 避谶、避旧、避人

3. **节气/时令式**
   - 惊蛰：低谷期，就是你的改命期
   - 春分：人生的转折点

4. **痛点直击式**
   - 一个家庭要向上走，一定要...
   - 发现一个奇怪的现象...

## 📐 文章结构模板

### 标准结构（3000-5000字）

```
开篇引入（200-300字）
    ↓
01 分论点1（800-1200字）
    ↓
02 分论点2（800-1200字）
    ↓
03 分论点3（800-1200字）
    ↓
结尾升华（150-200字）
```

## ✍️ 开篇模板

```markdown
【名人名言/古语引用】

【点明主题，1-2句话】

【提出核心观点，2-3句话】

【过渡到正文，1句话】
```

### 示例

```
南怀瑾先生曾说："家庭者，人生之锚也。
锚定，则船行万里不迷航；锚失，则风浪一至即倾覆。"

一个家庭若想向上走，不在外求而在内守。

细究起来无非六个字：避谶，避旧，避人。
```

## 📖 正文模板（每节）

### 结构

```markdown
## 0X 【小标题：简洁有力】

【引入语/过渡句】

【反面案例】
- 具体场景描写（2-3段）
- 人物对话/细节
- 结果/后果

【正面案例】
- 具体场景描写（2-3段）
- 人物对话/细节
- 结果/成就

【名言/金句总结】

【升华/过渡到下一节】
```

### 小标题格式

**格式**：分论点 + 行动指南

**示例**：
- 避谶，管住嘴
- 避旧，放下怨
- 避人，收拢心
- 蛰伏，是为了更有力的爆发
- 破土，要找准时机与方向

## 🎭 案例使用规则

### 案例类型

1. **名人案例**（40%）
   - 刀郎、阿来、林清玄
   - 曾国藩、郭晶晶、霍启刚
   - 艾伦·莱恩、庞众望

2. **文学影视案例**（30%）
   - 《忠诚》、《人世间》、《一句顶一万句》
   - 《曾国藩家书》、《菜根谭》、《马桥词典》

3. **身边故事**（30%）
   - "我有个朋友..."
   - "不久前的家庭聚会..."
   - "我认识一位前辈..."

### 正反对比法（必须使用）

```
1. 先讲反面案例（失败、痛苦、错误）
   - 具体场景
   - 人物对话
   - 悲惨结果

2. 再讲正面案例（成功、幸福、正确）
   - 具体场景
   - 人物对话
   - 美好结果

3. 形成强烈对比，增强说服力
```

### 案例写作要点

- ✅ 细节丰富，场景具体
- ✅ 有人物对话
- ✅ 有情感描写
- ✅ 有结果呈现
- ❌ 避免空洞说教
- ❌ 避免过度煽情

## 💎 金句使用规则

### 金句位置（必须）

1. **开篇第一句**：名人名言/古语
2. **每节开头**：引出分论点
3. **每节结尾**：总结升华
4. **文章结尾**：点题升华

### 金句类型

#### 1. 古语类
```
- "伏久者飞必高，开先者谢独早。"
- "昨日之非不可留，留之则根荄复萌。"
- "潜龙勿用，阳在下也。"
```

#### 2. 哲理类
```
- "真正的成长，也从来不在人声鼎沸的高处，而在无人问津的低谷。"
- "一个人最值得炫耀的东西，不是金钱，不是家世，而是教养、善良、努力向上的品质。"
```

#### 3. 行动类
```
- "你只管负责把根扎稳，把低谷时该吃的苦、该经受的历练都一一承担。"
- "从今天起，管住嘴，放下怨，收拢心。"
```

### 金句创作技巧

1. **对仗工整**
   - 管住嘴，放下怨，收拢心
   - 避谶，避旧，避人

2. **排比增强**
   - 不说丧气话，不贴负面标签，别让负面情绪困住自己和家人

3. **短句有力**
   - 沉潜日久，必有腾飞时
   - 以何眼观世界，观到何种世界

## 🎬 结尾模板

```
▽

【古语/诗词引用】

【总结核心观点，2-3句话】

【情感升华，2-3句话】

【行动号召，1-2句话】

点个推荐，与朋友们共勉。
```

**⚠️ 引导语不要写具体账号名或二维码描述**，这部分由运营者自行替换。

### 示例

```
▽

古语云：家运之盛衰，天不能操其权，而其自操之。

家运从来不在别处，而在你我的言行与选择里。

从今天起，管住嘴，放下怨，收拢心。

当你开始守护家的能量，家自会回馈你向上的力量。

点个推荐，与朋友们共勉。
```

**⚠️ 引导语不要写具体账号名或二维码描述**，这部分由运营者自行替换。

## 🎨 语言风格要求

### 句式特点

1. **短句为主**（70%）
   - 每句10-20字
   - 节奏明快
   - 易于理解

2. **排比句**（20%）
   - 增强气势
   - 加深印象
   - 便于记忆

3. **长句**（10%）
   - 用于描写场景
   - 用于情感铺垫

### 修辞手法

1. **比喻**
   - "母亲积极的话语，将日子的苦化为生活的糖"
   - "家是一个人最终的归宿，也是所有幸福的源头"

2. **引用**
   - 名人名言
   - 古语诗词
   - 文学作品
   - 专家观点

3. **对比**
   - 正面 vs 反面
   - 过去 vs 现在
   - 穷人思维 vs 富人思维

### 段落要求

- **短段落**：每段2-4句话
- **空行分隔**：段落之间有明显空行
- **视觉舒适**：避免大段文字

## 📊 数据规范

### 文章长度（⚠️ 此处覆盖 SKILL.md 默认的 1200-1500 字要求）

鸡汤文传播力强依赖情感深度与案例厚度，**专用字数标准如下：**
- **总字数**：3000-5000 字
- **开篇**：200-300 字
- **每节**：800-1200 字
- **结尾**：150-200 字

### 段落规范
- **每段**：2-4句话
- **每句**：10-30字
- **空行**：段落之间必须有空行

### 标点使用
- **句号**：短句结束
- **冒号**：引出案例或观点
- **引号**：突出关键词或对话
- **顿号**：并列关系

---

## 🎭 情感基调

### 整体基调
- **温暖励志**：传递正能量，给人希望
- **理性克制**：不煽情，不鸡汤，有深度
- **共鸣感强**：贴近生活，引发思考

### 情感节奏

```
开篇：平静引入（名言 + 观点）
  ↓
反面案例：情绪低落（痛点 + 共鸣）
  ↓
正面案例：情绪上扬（希望 + 启发）
  ↓
金句总结：情绪高潮（共鸣 + 力量）
  ↓
结尾升华：平静而坚定（行动 + 号召）
```

## 🔧 写作流程

### 步骤1：确定主题
- 选择一个核心主题
- 提炼核心观点（一句话）
- 确定3个分论点

### 步骤2：搜集素材
- 搜集3-6个案例（正反各半）
- 整理相关名言金句
- 准备开篇和结尾的古语

### 步骤3：组织结构
- 按模板组织文章结构
- 安排案例顺序（先反后正）
- 设计小标题

### 步骤4：撰写初稿
- 开篇：名言 + 观点
- 正文：案例 + 金句
- 结尾：古语 + 升华

### 步骤5：打磨语言
- 提炼金句
- 缩短句子
- 增加排比
- 检查对仗

### 步骤6：设计标题
- 使用标题公式
- 制造冲突感
- 突出痛点
- 给出方案
