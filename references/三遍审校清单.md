# 三遍审校清单

文章初稿写完后，**强制按以下三遍顺序逐句审校**。每一遍都要逐句过、标记问题、说明修改理由。

> 风格规则与替换词表详见 [HUMANIZE_GUIDE.md](../HUMANIZE_GUIDE.md)。本清单只规定"过几遍、查什么、勾什么"，不重复列出风格规则。

---

## 第一遍：内容审校（逻辑、事实、结构）

### 事实准确性
- [ ] 所有数据都有真实来源（不编造）
- [ ] 时间正确（年份、月份、顺序）
- [ ] 产品名 / 人名 / 公司名 / 版本号正确
- [ ] 技术术语使用准确
- [ ] 链接有效可访问

**铁律：宁可少写，不要编造。** 不确定的事实 → 删掉，或调用搜索工具核实。

### 逻辑清晰性
- [ ] 前后无矛盾（前面说 A，后面不能说非 A）
- [ ] 因果合理（不要"因为用了 AI 所以成功了"这种过度简化）
- [ ] 论据支持论点（一个例子不下普遍性结论）

### 结构合理性
- [ ] 小标题概括段落内容
- [ ] 各部分长度均衡，无某段独占一半
- [ ] 不跑题
- [ ] 重要信息前置

---

## 第二遍：风格审校（降 AI 味）⭐ 核心

按 [HUMANIZE_GUIDE.md](../HUMANIZE_GUIDE.md) 的规则改稿，但本遍只用下面的 **6 项 checklist 验收**：

- [ ] **删除 AI 套话**：全文搜索"在当今 / 随着 / 值得注意的是 / 综上所述 / 让我们 / 在...的背景下 / 毋庸置疑 / 不言而喻"——一个不留
- [ ] **拆解工整句式**：「A 不对 B 才对」句式整篇最多用 1 次；不出现连续的"既...又... / 不仅...而且... / 从 A 到 B，从 C 到 D"
- [ ] **替换书面词汇**：「显著 / 充分 / 进行 / 实施 / 开展 / 大幅 / 切实 / 行之有效」全部替换为口语化表达
- [ ] **加入 ≥ 10 处个人感受 / 吐槽 / 态度词**：「我觉得 / 说实话 / 我用下来 / 挺好用 / 贵死了 / 离谱」之类
- [ ] **抽象 → 具体**：找出"很好 / 很强 / 显著"等形容词，能换数字 / 场景就换
- [ ] **大声朗读**：听起来像和朋友聊天，不是论文

### 🚨 替换词反复使用元规则（重要）

删 AI 套话之后最容易掉进的新坑：用同一个口语词替换所有套话，结果"说白了"满篇飞，反而出现新的 AI 味。

**单文限频规则（硬性）：**
- 「说白了」整篇 ≤ 1 次
- 「为什么？」整篇 ≤ 1 次（优先直接用"原因是.../就是因为..."替代）
- 「很清楚 / 非常清楚」直接删除，不替换
- 「重点来了」整篇 ≤ 1 次

替换时主动使用 **多种** 表达组合，保持自然变化：
- 「值得注意的是」→ 可换"注意 / 重点来了 / [直接说内容]"，**不要每次都用同一种**
- 「综上所述」→ 可换"总结一下 / 说白了 / 简单来说 / [直接给结论]"

## 第三遍：细节打磨（标点、排版、节奏）

### 句子 / 段落
- [ ] 主流句子 15-25 字
- [ ] **没有 > 30 字的句子**（超出全部拆短）
- [ ] 段落 3-5 行 / 100-150 字（手机屏幕一屏）
- [ ] 长短句交替，节奏不单调

### 标点（公众号尤其重要）
- [ ] 所有逗号 = 全角「，」
- [ ] 所有句号 = 全角「。」
- [ ] 所有冒号 = 全角「：」
- [ ] **所有引号 = 「」**（不是 "" 或 ''；双重引号用 『「」』）
- [ ] 多用句号，少用逗号连接长句
- [ ] 感叹号一段最多 1-2 个

### 中英文混排（强制）
公众号读者在手机上看，混排不规范会非常显眼。

**规则：**
1. 中文 + **空格** + 英文 / 数字
2. 英文 / 数字 + **空格** + 中文
3. 英文 / 数字 + 标点（**无空格**）+ 中文
4. 中文 + 标点（**无空格**）+ 英文 / 数字

**示例：**
- ✅ 「我用 Cursor 做了一个项目」
- ✅ 「AI 功能使用量涨了 10 倍」
- ✅ 「获得了 5000 美金（约 3.5 万人民币）」
- ❌ 「我用Cursor做了一个项目」（缺空格）
- ❌ 「我用 Cursor 、 Claude Code」（标点前多余空格）

**checklist：**
- [ ] 中文与英文之间有空格
- [ ] 中文与数字之间有空格
- [ ] 标点符号前后无多余空格
- [ ] 全文使用中文标点（不是英文逗号 , / 句号 .）

---

## 🎯 完成审校后的终极检查

- [ ] 大声朗读全文，听起来自然
- [ ] 删了 AI 套话之后，没有"说白了 / 为什么 / 很清楚"满篇飞的新 AI 味
- [ ] 中英文空格全部规范
- [ ] 引号全部 「」
- [ ] 句子全部 ≤ 30 字
- [ ] 至少 10 处个人态度表达
- [ ] 数据全部真实，编造的全部删除

**未通过任意一项 → 不得进入下一步（生成封面 / 创建草稿）。**
